# FCK c&p - The Simplest Way to Excel

### The Problem
Manually copy-pasting columns between Excel files is tedious, error-prone, and frustrating. One slip of the mouse and your data is misaligned.

### The Solution
**FCK c&p** is a secure, serverless tool that automates Excel column mapping. 
1. Drag & Drop your Source file.
2. Drag & Drop your Target file.
3. Define simple rules (e.g., "Take Column A from Source and put it in Column B of Target").
4. **Done.**
